v0.4.2 · darwin/arm64

Imagined CLI surface for Facies — what it looks like when you run facies from a terminal. Same visual idiom as the dashboard: amber for keys and emphasis, cyan for identifiers, green for monetary values, ASCII glyphs for sparklines and stratigraphy bars. Use these as reference for README screenshots, docs, and demo recordings.
